Nobel Prize Database Search

My group's end project for CMPT 305 built using Java and JavaFX. It pulls a CSV off of http://api.nobelprize.org/ and parses it into our database class structure. From there, we use JavaFX to create a GUI for the user. Personally, I took care of the UI design for the main page. Things I implemented are the following:

  • Design for the Main Screen's UI.
  • Use of JavaFX's threading to create the clickable objects to keep the UI functional during long searches.
  • Use of animation for the sliding search bar to create a familiar website look to keep it user friendly.
  • Using the third-party ControlsFX library to have a double-headed slider for Year
  • Creating buttons which showed some of the information about the laureate and interacting with our database's information

Template E-mails Outlook Add-In

A proof of concept Outlook Add-In produced in C# which allowed for loading and saving e-mails as a "Template". This would store and load e-mail messages under the ".oft" extension in the user's My Documents/Outlook Templates folder. If the folder was not created, it would create the folder on the add-in's startup. The Templates Drop-Down Box will be populated only by ".oft" files.

The thought behind this program was to help improve my productivity while working in an IT setting. I found that I sent many similar e-mails to users, and wanted to have a one click solution to save and pull up e-mails to send to another user.
