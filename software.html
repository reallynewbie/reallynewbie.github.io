Template E-mails Outlook Add-In

A proof of concept Outlook Add-In produced in Visual Studio 2010 which allowed for Loading and Saving E-mails as a "Template". This would store and load e-mail messages under the ".oft" extension in the user's My Documents/Outlook Templates folder. If the folder was not created, it would create the folder on the add-in's startup. The Templates Drop-Down Box will be populated only by ".oft" files.

The thought behind this program was to help improve my productivity while working in an IT setting. I found that I sent many similar e-mails to users, and wanted to have a one click solution to save and pull up e-mails to send to another user.
